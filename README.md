# Poi~, dotNET #

## 简介 ##

![Title](assets/title.jpg)

《Poi~, dotNET》是一份有关于 Dot Net 技术栈的理论性的文档合集，它不涉及任何 dotNET 的应用与实践，你可将本合辑看成一份适合培训团队新人的基础教程，或是一份关于填补基础缺憾的填坑手册，或是一份可用于查阅 Dot Net 底层实现原理的指南。
